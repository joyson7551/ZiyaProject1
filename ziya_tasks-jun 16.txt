TASK 1
======
Create a new API route in Express.

Endpoint: POST /api/posts

Functionality Requirements:

Accept form data containing title, content, and tags.

Validate the data to ensure no fields are left empty.

Save the post to MongoDB using Mongoose.

Return a success response upon successful creation.

If you have any questions or need clarification, please feel free to ask.

Task: Build a “Create Post” page* for the blog project.

Requirements:

. Title, Content, Tags (comma-separated) input fields.
